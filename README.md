ABOUT THE PROJECT
## About The Project

 Love&Care is program to deliver stray animals a good life, good food and care. We are here to provide a platform to establishing events for Animal Welfare.

### Built With

* Node.JS
*	Express
*	Node Package Manager
*	Mongodb
*	Mongoose
*	Javascript
*	Template engine.
* DOM manipulation.
*	HTML-5
*	CSS/Bootstrap
*	Handlebar
*	Express file-upload.
*	Cloudinary
*	Heroku
* Nodemailer

1. Clone the repo
   ```sh
   git clonehttps://github.com/Nithya-B-AU16/loveNcare.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. npm start
